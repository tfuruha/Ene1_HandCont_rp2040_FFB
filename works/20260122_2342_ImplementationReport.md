RP2040 移行初期実装 ウォークスルー
RP2040への移行に向けた基盤実装（shared_data.h / main.cpp）を完了しました。

実装内容
1. Core間データ共有 (
shared_data.h
)
Core 1 (制御) から Core 0 (通信) へデータを渡すための SharedData 構造体を定義しました。

ステアリング角度、アクセル、ブレーキ、ボタン状態の保持。
将来的な FFB 受信用の目標トルク・ステータス変数を先行実装。
volatile キーワードによりコア間での可視性を確保。
2. マルチコア・スケルトン (
main.cpp
)
Core 0:
Adafruit_TinyUSB を使用した HID 通信の初期化。
20ms周期でのホストへのデータレポート送信。
Core 1:
1ms (1000us) 周期の厳格なタイマー制御を micros() で実装。
ドリフトを防ぐため、実行時刻を加算する方式を採用。
動作確認用として 1000ms ごとの LED 点滅ロジックを配置。
